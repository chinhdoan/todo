#TO

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install package.

```sh
$ npm install
```

## Run web

For production

```bash
$ npm run build
```

For development

```bash
$ npm run server
```

## Usage

After run server , You can find it at [localhost:8080](localhost:8080)
